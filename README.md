# distilatex

> A LaTeX summarizer, that extracts all theorems & definitions from a .tex file to quickly go over handouts

## Installation

```sh
cargo install distilatex
```

## Usage

```sh
distilatex my_horribly_long_handout.tex > summary.tex
```
